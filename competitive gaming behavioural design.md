## Context

I'm a former Team Fortress 2 player, and I've been playing Overwatch since 2018. One of the first things I noticed about Overwatch was that unlike TF2, the server browser was hidden away and not a part of the core game loop: the front-facing modes have always been Unranked (formerly Quick Play) and Competitive. This has always bothered me, and since I started studying the human mind, I've begun to see the wider implications of this design choice. 

This is where I'll talk about it.
